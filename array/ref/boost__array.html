Description

This class represents a container that encapsulates fixed size arrays. It has the same semantics as a struct holding a C-style array T[N]. Unlike a C-style array, it doesn't decay to T* automatically. The class combines the performance of a C-style array with the convenience of a standard container. There is a special case for a zero-length array (N == 0). In that case, array.begin() == array.end(), which is some unique value.
