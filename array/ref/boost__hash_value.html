boost::hash_value

Calculate hash value for an array.

Synopsis

Defined in header <boost/array.hpp> 

template<
    class T,
    std::size_t N>
std::size_t
hash_value(
    const array< T, N >& arr);
Description

This function calculates the hash value for an array

Complexity

Linear

Remarks

This function is used to calculate the hash for an array, which allows it to be inserted in unordered containers.

Return Value

hash value for an array

Parameters

Name

Description

arr

Array for which we calculate the hash
