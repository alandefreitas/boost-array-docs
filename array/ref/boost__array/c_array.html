c_array

[indexterm2 c_array..]

Direct access to the underlying array.

Synopsis 
T*
c_array();
Description

Returns pointer to the underlying array serving as element storage.

Complexity

Constant

Remarks

The pointer is such that range [data(); data() + size()) is always a valid range, even if the container is empty. When the container is empty, data() is not dereferenceable. If size() is 0, data() may or may not return a null pointer. This function is not available in std::array

Return Value

Pointer to the underlying element storage For non-empty containers, the returned pointer compares equal to the address of the first element.
