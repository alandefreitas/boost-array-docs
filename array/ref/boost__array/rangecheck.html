rangecheck

[indexterm2 rangecheck..]

Check range.

Synopsis 
static
constexpr bool
rangecheck(
    size_type i);
Description

range This function checks if i in the valid range for the array. This function may be private because it is static.

Complexity

Constant

Exception Safety

Throws std::out_of_range if i is not in the valid range.

Remarks

This function is not available in std::array

Return Value

True if i is in the valid range for the array
