front (2 of 2 overloads)

Access the first array element.

Synopsis 
constexpr const_reference
front() const;
Description

This function returns a reference to the first element in the container. Calling front on an empty container is undefined. This overload gets called when this is constant.

Complexity

Constant

Remarks

c.front()is equivalent to *c.begin()

Preconditions

!(empty())

Return Value

Reference to the first array element.
