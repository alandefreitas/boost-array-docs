assign

[indexterm2 assign..]

Assign one value to all elements.

Synopsis 
void
assign(
    const T& value);
Description

This function assigns the same value to all arrays elements

Complexity

Linear

Remarks

This function is not available in std::array. This function is a synonym for fill.

Return Value

(none)

See Also

See Also operator=, fill
