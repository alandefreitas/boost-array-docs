crbegin

[indexterm2 crbegin..]

Returns a reverse iterator to the beginning.

Synopsis 
const_reverse_iterator
crbegin() const;
Description

Returns a reverse iterator to the first element of the reversed array. This reverse iterator corresponds to the last element of the non-reversed array. This function is usually called to provide a constant iterator when this is not constant.

Complexity

Constant

Remarks

If the array is empty, the returned iterator is equal to rend().

Return Value

Reverse iterator to the first element of the reversed array
