at (2 of 2 overloads)

Access specified element with bounds checking.

Synopsis 
const_reference
at(
    size_type i) const;
Description

This function Returns a reference to the element at specified location i, with bounds checking. If pos is not within the range of the container, an exception of type std::out_of_range is thrown. This overload gets called when this is constant.

Complexity

Constant

Preconditions

!(i < size())

Exception Safety

std::out_of_rangeif !(i < size()).

Return Value

Reference to the requested element.

Parameters

Name

Description

i

Position of the element to return
