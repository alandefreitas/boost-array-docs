rbegin (1 of 2 overloads)

Returns a reverse iterator to the beginning.

Synopsis 
reverse_iterator
rbegin();
Description

Returns a reverse iterator to the first element of the reversed array. This reverse iterator corresponds to the last element of the non-reversed array.

Complexity

Constant

Remarks

If the array is empty, the returned iterator is equal to rend().

Return Value

Reverse iterator to the first element of the reversed array
