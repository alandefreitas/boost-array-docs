swap

[indexterm2 swap..]

Swaps the container contents.

Synopsis 
void
swap(
    array< T, N >& y);
Description

Swap Exchanges the contents of the container with those of other. This function does not cause iterators and references to associate with the other container.

Complexity

Linear

Remarks

Because array<T, N> is not dynamically allocated, this operation has linear complexity.

Return Value

(none)

Parameters

Name

Description

y

container to exchange the contents with
